HERO SLIDER IMAGES
==================

Drop three files into this folder, named exactly:

  slide-1.jpg
  slide-2.jpg
  slide-3.jpg

Recommended size: 2000 x 1200 px or larger, landscape orientation.
Compress to under ~250 KB each for fast loading.

If you delete this README, nothing breaks. The homepage will still work.

If the slide-X.jpg files are missing, the hero falls back to stock
cycling photos so the site never appears broken while you're working.
