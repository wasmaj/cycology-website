EXECUTIVE & TRUSTEE HEADSHOTS
=============================

Drop headshot photos into this folder using a kebab-case slug for the
filename — first-last.jpg — matching the src referenced on the page.

Currently expected files for /pages/club-executives.html:

  kemi-ajayi.jpg               (Kemi Ajayi — Finance Secretary)
  ogbonnaya-obi.jpg            (Ogbonnaya Obi — Membership Secretary)
  ralph-okoro.jpg              (Ralph Okoro — Media & Publicity Secretary)
  dolu-olugbenjo.jpg           (Dolu Olugbenjo — CSR Secretary)
  ayodimeji-olugbewesa.jpg     (Ayodimeji Olugbewesa — VC Internal Rides)

Recommended specs:
  - Portrait orientation, 800 × 1000 px (4:5 aspect ratio)
  - Compressed under ~200 KB each (use Squoosh.app or similar)
  - JPG or WebP

Until a real photo is uploaded, each card automatically shows a brand-
coloured placeholder with the person's initials. No code edit needed.

To add a new executive or trustee:
  1. Open /pages/club-executives.html (or /pages/board-of-trustees.html)
  2. Copy any <article class="team-card"> block
  3. Update the name, role, image src and data-initials attribute
  4. Save your photo here using the matching filename
